## 汇报结构自检清单

### 金字塔原理检查

- [ ] **结论先行** - 开头就给出了核心结论
- [ ] **以上统下** - 上层思想是下层的概括
- [ ] **归类分组** - 思想按逻辑关系分组
- [ ] **逻辑递进** - 每组思想按顺序排列

### SCQA 检查

- [ ] **S** - 情境是共识事实
- [ ] **C** - 冲突与情境形成对比
- [ ] **Q** - 问题自然引出
- [ ] **A** - 回答直接回应问题

### TOPS 原则检查

- [ ] **T（Target）** - 目标明确，有的放矢
- [ ] **O（Overall）** - 结论先行，上下贯通
- [ ] **P（Power）** - 数据支撑，有理有力
- [ ] **S（Structure）** - 结构清晰，逻辑递进

### 表达检查

- [ ] 语言简洁，无冗余
- [ ] 数据准确，有来源
- [ ] 逻辑清晰，无跳跃
- [ ] 格式美观，易阅读
